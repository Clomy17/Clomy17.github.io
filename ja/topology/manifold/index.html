ℝ𝑛から見た位相空間

定義

多様体

局所座標系

位相空間(𝑀,𝒪)を与える．

𝑀の局所座標系(local chart)またはチャートとは以下のデータからなる：

  • 𝑀の開集合𝑈∈𝒪
  • 𝑛∈ℕに対し，同相写像 𝜑 : 𝑈 → ℝ𝑛

これらは以下を満たす：

局所座標
  • 位相空間(𝑀,𝒪)
  • 局所座標系(𝑈,𝜑)

を与える．

  • ∀⁡𝑝∈𝑈 に対し，
    • 𝑈を𝑝の座標近傍(coordinate neighborhood)と呼ぶ．
    • 𝜑 ⁡ (𝑝) ∈ ℝ𝑛 を 𝑝の局所座標(local coordinate)と呼ぶ．
  • ℝ𝑛 の第 𝑖 成分への射影演算子 𝜋𝑖 に対し， 𝑥𝑖 ≔ 𝜋𝑖 ∘ 𝜑 を 𝑈 上の座標函数と呼ぶ．

多様体

𝑛次元位相多様体(manifold)とは以下のデータからなる：

  • Hausdorff空間(𝑀,𝒪)

これらは以下を満たす：

  • 任意の点 𝑝∈𝑀 に対し， 𝑝 の 近傍𝑈∈𝒪 と ℝ𝑛 への同相写像 𝜑 : 𝑈 → ℝ𝑛 との組が存在し，これが局所座標系をなす．
