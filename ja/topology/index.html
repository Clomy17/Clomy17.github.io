位相空間の定義

同値な定義が割とあるので一通り示しておくが，基本は開集合系で定義する．

位相空間

位相空間(topological space)とは，以下の2つのデータからなる．

  • 集合𝑋
  • 以下のいずれかのデータを持つ:
    • 位相(topology)もしくは開集合系と呼ばれる𝑋冪集合の部分集合 𝒪 ⊂ 2𝑋
    • 閉集合系と呼ばれる𝑋の冪集合の部分集合 𝒪 c ⊂ 2𝑋
    • 𝑋の部分集合の閉包(clousure)をとる操作 cl : 2𝑋 → 2𝑋
    • 𝑋の部分集合の内部(interior)をとる操作 int : 2𝑋 → 2𝑋
    • 𝑋の部分集合の外部(exterior)をとる操作 ext : 2𝑋 → 2𝑋
    • 𝑋の部分集合の境界(boundary)をとる操作 ∂ : 2𝑋 → 2𝑋
    • 𝑋の部分集合の導（来）集合(derived set)をとる操作 -∗ : 2𝑋 → 2𝑋
    • 𝑋の各元の近傍系(neighbourhood system)をとる操作 𝑁 : 𝑋 → 2 2𝑋
    • 有向点族の収束(convergence of nets)
    • フィルターの収束(convergence of filters)

それぞれに対し，本質的に等価な以下の条件を満たす．

点・開集合・閉集合・内点・外点・境界点・集積点

位相空間 ( 𝑋,𝒪 ) を与える．

  • 𝑋 の元を点(point)と呼ぶ．
  • 開集合系 𝒪 の元のことを開集合(open set)と呼ぶ．
  • 閉集合系 𝒪c の元のことを閉集合(closed set)と呼ぶ．
  • ∀⁡𝐴⊂𝑋に対し，𝐴 の内部 int⁡(𝐴) の元のことを 𝐴 の内点(interior point)と呼ぶ．
  • ∀⁡𝐴⊂𝑋に対し，𝐴 の外部 ext⁡(𝐴) の元のことを 𝐴 の外点(exterior point)と呼ぶ．
  • ∀⁡𝐴⊂𝑋に対し，𝐴 の境界 ∂⁡(𝐴) の元のことを 𝐴 の境界点(boundary point)と呼ぶ．
  • ∀⁡𝐴⊂𝑋に対し，𝐴 の導集合 𝐴∗ の元のことを 𝐴 の集積点(accumulation point)と呼ぶ．
  • ∀⁡𝑥∈𝑋に対し，𝑥 の近傍系 𝑁⁡(𝑥) の元のことを 𝑥 の近傍(neighbourhood)と呼ぶ．

位相空間の公理の等価性

位相空間の各公理は，以下のような対応により本質的に等価：

  • 𝒪が開（閉）集合系の公理を満たす⇔𝒪の補集合の集合𝒪cが閉（開）集合系の公理を満たす
  • 開集合系 𝒪 について， 𝑁 : 𝑋 → 2 2𝑋 , 𝑥 ↦ { 𝑈 ∈ 2𝑋 | ∃ ⁡ 𝑉∈𝒪 , ( 𝑥∈𝑉 , 𝑈⊂𝑉 ) } は 𝑥 の近傍系の公理を満たす．
  • int ⁡ ( 2𝑋 ) , ext ⁡ ( 2𝑋 ) は開集合系の公理を満たし， cl ⁡ ( 2𝑋 ) は閉集合系の公理を満たす．
  • ∀⁡𝐴⊂𝑋に対し，𝐴を含む最大の開集合を取る操作intは，内部の公理を満たす．
  • ∀⁡𝐴⊂𝑋に対し，𝐴を含まない最大の開集合を取る操作extは，外部の公理を満たす．
  • ∀⁡𝐴⊂𝑋に対し，𝐴を含む最小の閉集合を取る操作clは，閉包の公理を満たす．
  • cl∖int,cl∖ext は境界作用素 ∂ の公理を満たす．
  • id∪-∗ は閉包作用素 cl の公理を満たす．

条件の同値性の証明

  • 開集合系，閉集合系による定義の同値性は，de Morganの法則より成り立つ．

位相空間の圏

位相空間の圏(category of topological spaces) 𝐓𝐨𝐩 とは，以下で定まる圏のこと：

  • 対象を全ての位相空間
  • 射を全ての連続写像

同相写像

位相空間 ( 𝑋 , 𝒪𝑋 ) , ( 𝑌 , 𝒪𝑌 ) を与える．

連続写像 𝑓:𝑋→𝑌 が以下を満たすとき，𝑓 を同相(homeomorphism)と呼ぶ：

  • 𝑓が全単射
  • 𝑓の逆写像𝑓−1:𝑌→𝑋が連続

𝑋と𝑌のことも同相(homeomorphic)と呼び，𝑋≃𝑌と表す．
