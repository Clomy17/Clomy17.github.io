位相空間

定義

位相空間

位相空間(topological space)とは，以下の2つのデータからなる．

  • 集合𝑋
  • 位相(topology)と呼ばれる冪集合の部分集合 𝒪 ⊂ 2 𝑋

これらは以下を満たす．

  • ∅ , 𝑋 ∈ 𝒪
  • ∀ ⁡ 𝑛 ∈ ℕ に対して以下を満たす ∀ ⁡ 𝑖 ∈ ℕ ≤ 𝑛 , 𝑂 𝑖 ∈ 𝒪 ⇒ ⋂ 𝑖 = 1 𝑛 𝑂 𝑖 ∈ 𝒪
  • 任意の集合𝛬に対して以下を満たす ∀ ⁡ 𝜆 ∈ 𝛬 , 𝑂 𝜆 ∈ 𝒪 ⇒ ⋃ 𝜆 ∈ 𝛬 𝑂 𝑖 ∈ 𝒪

開集合

位相空間 ( 𝑋 , 𝒪 ) を与える．

  • 𝒪の元のことを開集合(open set)と呼ぶ．
  • 𝑂∈𝑋で，𝑋∖𝑂が開集合となるものを閉集合(closed set)と呼ぶ．

De Morganの法則より，位相空間は閉集合を用いても定義可能：

閉集合系による位相空間

位相空間(topological space)とは，以下の2つのデータからなる．

  • 集合𝑋
  • 閉集合系 𝒪 c ⊂ 2 𝑋

これらは以下を満たす．

  • ∅ , 𝑋 ∈ 𝒪 c
  • ∀ ⁡ 𝑛 ∈ ℕ に対して以下を満たす ∀ ⁡ 𝑖 ∈ ℕ ≤ 𝑛 , 𝑂 𝑖 ∈ 𝒪 c ⇒ ⋃ 𝑖 = 1 𝑛 𝑂 𝑖 ∈ 𝒪 c
  • 任意の集合𝛬に対して以下を満たす ∀ ⁡ 𝜆 ∈ 𝛬 , 𝑂 𝜆 ∈ 𝒪 c ⇒ ⋂ 𝜆 ∈ 𝛬 𝑂 𝑖 ∈ 𝒪 c
