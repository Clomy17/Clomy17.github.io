マグマ上の加群

定義

マグマ上の左加群

マグマ𝐺を与える．

（マグマ上の）左𝐺-加群とは，以下のデータからなる．

  • 可換群𝑀
  • スカラー乗法と呼ばれる演算 𝐺 × 𝑀 → 𝑀

これらは以下の条件を満たす．

  • ∀ ⁡ 𝑔 ∈ 𝐺 , ∀ ⁡ 𝑥 , 𝑦 ∈ 𝑀 , 𝑔 ⁢ ( 𝑥 + 𝑦 ) = 𝑔 ⁢ 𝑥 + 𝑔 ⁢ 𝑦
  • ∀ ⁡ 𝑔 , ℎ ∈ 𝐺 , ∀ ⁡ 𝑥 ∈ 𝑀 , ( 𝑔 ⁢ ℎ ) ⁢ 𝑥 = 𝑔 ⁢ ( ℎ ⁢ 𝑥 )

マグマ上の右加群

マグマ𝐺を与える．

（マグマ上の）右𝐺-加群とは，以下のデータからなる．

  • 可換群𝑀
  • スカラー乗法と呼ばれる演算 𝑀 × 𝐺 → 𝑀

これらは以下の条件を満たす．

  • ∀ ⁡ 𝑥 , 𝑦 ∈ 𝑀 , ∀ ⁡ 𝑔 ∈ 𝐺 , ( 𝑥 + 𝑦 ) ⁢ 𝑔 = 𝑥 ⁢ 𝑔 + 𝑦 ⁢ 𝑔
  • ∀ ⁡ 𝑥 ∈ 𝑀 , ∀ ⁡ 𝑔 , ℎ ∈ 𝐺 , 𝑥 ⁢ ( 𝑔 ⁢ ℎ ) = ( 𝑥 ⁢ 𝑔 ) ⁢ ℎ

マグマ𝐺上の左𝐺-加群とその反対マグマ 𝐺 op 上の右𝐺-加群は一対一に対応する．よって以下では基本的に左加群を考える．

特に可換マグマの反対環は自分自身に同型だから，これらの区別は記法の問題のみになる:

可換マグマ上の加群

𝐺が可換マグマの場合， 𝑟 ⁢ 𝑥 = 𝑥 ⁢ 𝑟 と対応させることで左𝐺-加群と右𝐺-加群が一対一に対応する．よって，このとき（特に左𝐺-加群を指して）単に𝐺-加群と呼ぶ．

マグマ上の加群準同型

  • マグマ𝐺
  • 左𝐺-加群𝑀,𝑁

を与える．

写像 𝑓 : 𝑀 → 𝑁 が ∀ ⁡ 𝑥 , 𝑦 ∈ 𝐺 , ∀ ⁡ 𝑥 , 𝑦 ∈ 𝑀 , 𝑓 ⁡ ( 𝑔 ⁢ 𝑥 + ℎ ⁢ 𝑦 ) = 𝑔 ⁢ 𝑓 ⁡ ( 𝑥 ) + ℎ ⁢ 𝑓 ⁡ ( 𝑦 ) を満たすとき，𝑓を（左𝑅-加群）準同型や𝑅-線型写像と呼ぶ．

環上の加群同型

全単射な（左/右）𝑅-加群準同型を（（左/右）𝐺-加群）同型と呼ぶ．

左𝐺-加群同型をもつ左𝑅-加群𝑀,𝑁のことも（左𝑅-加群）同型呼ぶ．

𝐺-加群の圏

マグマ𝐺を与える．

  • 全ての左𝐺-加群の集まりを対象
  • 全ての左𝐺-加群準同型の集まりを射

とする圏を左𝐺-加群の圏と呼び，𝐺-𝐌𝐨𝐝と書く．

部分𝑅-加群と呼ぶ．

単純
