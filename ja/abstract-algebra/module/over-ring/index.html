𝑅を環，𝑀を可換群とする．環上の左𝑅加群とは，𝑀とスカラー乗法と呼ばれる演算 𝑅 × 𝑀 → 𝑀 の組であって，以下の条件を満たすもの．

  • ∀ ⁡ 𝑟 , 𝑠 ∈ 𝑅 , ∀ ⁡ 𝑥 ∈ 𝑀 ( 𝑟 + 𝑠 ) ⁢ 𝑥 = 𝑟 ⁢ 𝑥 + 𝑠 ⁢ 𝑥
  • ∀ ⁡ 𝑟 , 𝑠 ∈ 𝑅 , ∀ ⁡ 𝑥 ∈ 𝑀 ( 𝑟 ⁢ 𝑠 ) ⁢ 𝑥 = 𝑟 ⁢ ( 𝑠 ⁢ 𝑥 )
  • ∀ ⁡ 𝑥 ∈ 𝑀 1 𝑅 ⁢ 𝑥 = 𝑥
