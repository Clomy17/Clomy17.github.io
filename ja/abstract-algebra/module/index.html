可換群・加群

可換群の定義と加群

可換群

可換群(commutative group)またはAbel群(abelian group)とは，以下の4つのデータからなる．

  • 集合𝑀
  • 和（または積）と呼ばれる𝑀の2項演算 + : 𝑀 × 𝑀 → 𝑀
  • 単位元 0 ∈ 𝑀
  • 逆元を取る操作（𝐺上の変換 - − 1 : 𝑀 → 𝑀 , 𝑥 ↦ − 𝑥 ）

これらは以下の条件を満たす．

  • 積は可換．つまり， ∀ ⁡ 𝑥 , 𝑦 ∈ 𝑀 , 𝑥 ⁢ 𝑦 = 𝑦 ⁢ 𝑥
  • 積は結合的．つまり， ∀ ⁡ 𝑥 , 𝑦 , 𝑧 ∈ 𝑀 , 𝑥 + ( 𝑦 + 𝑧 ) = ( 𝑥 + 𝑦 ) + 𝑧
  • 𝑒∈𝐺は𝐺の両側単位元．つまり， ∀ ⁡ 𝑥 ∈ 𝑀 , 0 + 𝑥 = 𝑥 + 0 = 𝑥
  • ∀ ⁡ 𝑥 ∈ 𝑀 に対し， − 𝑥 は𝑥の両側逆元．つまり， 𝑥 + ( − 𝑥 ) = ( − 𝑥 ) + 𝑥 = 0

和

可換群𝐺を与える．

  • 𝑀上の有限列に対し ∑ 𝑖 = 1 𝑛 𝑥 𝑖 ……
  • 特に，𝑥または−𝑥の𝑛∈ℤ≥0個のコピー { ±𝑥 } ⨿ 𝑛 を有限列とみなしたものは，群𝑀としての𝑥の冪に等しい．それを 𝑥 𝑛 の代わりに， 𝑛 ⁢ 𝑥 と表す．これを𝑥の𝑛個の和と呼ぶ．

可換群の構造を調べる準備として，加群を導入する．

可換群への作用の分配律

  • 集合𝐺
  • 可換群𝑀
  • 𝑀への左𝐺-作用

を与える．

以下を満たすとき，左𝐺-作用は分配的と呼ばれる： ∀ ⁡ 𝑔 ∈ 𝐺 , ∀ ⁡ 𝑥 , 𝑦 ∈ 𝑀 , 𝑔 ⁢ ( 𝑥 + 𝑦 ) = 𝑔 ⁢ 𝑥 + 𝑔 ⁢ 𝑦

可換群の𝑛個の和は左ℤ-作用と見做せることに注意して，次を確認しよう：

可換群の和の性質

  • 可換群𝑀
  • 集合𝐺
  • 𝑀への上の分配的な左𝐺-作用

を与えると，以下が成り立つ：

  • 𝑀の和 ℤ × 𝑀 → 𝑀 は分配的
  • 𝑀がをもつとき，𝐺上の作用と和は可換： ∀ ⁡ 𝑔 ∈ 𝐺 , ∀ ⁡ 𝑛 ∈ ℕ , ∀ ⁡ 𝑥 ∈ 𝑀 , 𝑔 ⁢ ( 𝑛 ⁢ 𝑥 ) = 𝑛 ⁢ ( 𝑔 ⁢ 𝑥 )

……

可換群の和に関して分配的な（左）作用のことを（左）加群と呼ぶ．

しかし今見たように，任意の可換群は和に関するℤ-加群となり，任意の分配的な左𝐺-作用と可換だから，より一般に次の構造を考えることができる：

左加群

  • 集合𝐺
  • 環𝑅

を与える．

左加群(left G-module)とは以下のデータからなる：

  • 可換群𝑀
  • 可換群𝑀への左𝐺-作用
  • スカラー乗法と呼ばれる可換群𝑀への左𝑅-作用

これらは以下を満たす：

  • 左𝐺-作用は分配的： ∀ ⁡ 𝑔 ∈ 𝐺 , ∀ ⁡ 𝑥 , 𝑦 ∈ 𝑀 , 𝑔 ⁢ ( 𝑥 + 𝑦 ) = 𝑔 ⁢ 𝑥 + 𝑔 ⁢ 𝑦
  • 左𝑅-作用は分配的： ∀ ⁡ 𝑟 ∈ 𝑅 , ∀ ⁡ 𝑥 , 𝑦 ∈ 𝑀 , 𝑟 ⁢ ( 𝑥 + 𝑦 ) = 𝑟 ⁢ 𝑥 + 𝑟 ⁢ 𝑦
  • 左𝐺-作用と左𝑅-作用は可換： ∀ ⁡ 𝑔 ∈ 𝐺 , ∀ ⁡ 𝑟 ∈ 𝑅 , ∀ ⁡ 𝑥 ∈ 𝑀 , 𝑔 ⁢ ( 𝑟 ⁢ 𝑥 ) = 𝑟 ⁢ ( 𝑔 ⁢ 𝑥 )

右加群も同様に定義できる．

可換群はℤ-加群

可換群は和をスカラー乗法とする左ℤ-加群．

証明

可換群の和の性質より明らか．

一般の左加群を考える準備として，𝐺が一般の環で，和をスカラー乗法とする場合を考えるべきだろう．それらについては，環の付加的な構造ごとに分けて以下で取り扱う：

  • 環上の左加群
  • 線型空間

𝐺≠𝑅の場合の左𝐺-加群は，左𝑅-加群にさらに付加的な構造を持っている場合を考えていることに対応する．このときの左𝐺-加群を𝐺の表現と読んだりする．𝐺については，単なる集合を含めた任意の代数的構造でもよく，非常に多くの場合が考えられる．そのため，詳しいことは𝐺の構造ごとのページにまとめることにする．

可換群の性質

可換群はℤ-加群なので，可換群の性質のかなり多くは一般の左𝑅-加群から受け継がれる．そのため，証明等の詳細はしばしばそちらに依存する．
