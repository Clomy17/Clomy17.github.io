加群

作用される側を単なる集合から，可換群に特殊化したもの．もしくは線型空間の一般化．

集合上の加群

集合上の左加群

𝐺を集合とする．左𝐺-加群(left 𝐺-module)は，以下の2つのデータからなる．

  • 可換群𝑀
  • スカラー乗法と呼ばれる演算 𝐺 × 𝑀 → 𝑀

これらは以下を満たす．

  • ∀ ⁡ 𝑔 ∈ 𝐺 , ∀ ⁡ 𝑥 , 𝑦 ∈ 𝑀 , 𝑔 ⁢ ( 𝑥 + 𝑦 ) = 𝑔 ⁢ 𝑥 + 𝑔 ⁢ 𝑦

右加群も定義できるが，作用と同様，基本的に左加群を考える．

加群が暗に満たす公理

𝐺を集合，𝑀を左𝐺-加群とする．単位元 0 𝑀 ∈ 𝑀 に対し， 𝐺 ⁢ 0 𝑀 = 0 𝑀

証明

∀ ⁡ 𝑔 ∈ 𝐺 , 𝑔 ⁢ 0 𝑀 = 𝑔 ⁢ 0 𝑀 + 0 𝑀 = 𝑔 ⁢ 0 𝑀 + 𝑔 ⁢ 0 𝑀 − 𝑔 ⁢ 0 𝑀 = 𝑔 ⁢ ( 0 𝑀 + 0 𝑀 ) − 𝑔 ⁢ 0 𝑀 = 𝑔 ⁢ 0 𝑀 − 𝑔 ⁢ 0 𝑀 = 0 𝑀

準同型

𝐺を集合，𝑀,𝑁を左𝐺-加群とする．群準同型 𝑓 : 𝑀 → 𝑁 が 𝑓 ⁡ ( 𝑔 ⁢ 𝑥 ) = 𝑔 ⁢ 𝑓 ⁡ ( 𝑥 ) を満たすとき，（左𝐺-加群）準同型((left 𝐺-module)homomorphism)と呼ぶ．

積の記号を省略しているためわかりづらいが，𝑓は単に𝑀の元を𝑁に送るだけで無く，𝑀のスカラー乗法も，𝑁のスカラー乗法に送っている．

同型

全単射な集合上の左加群準同型を（左𝐺-加群）同型((left 𝐺-module) isomorphism)と呼ぶ．

部分加群

𝐺を集合，𝑀を左𝐺-加群とする．𝑀の部分群𝑁が， 𝐺 ⁢ 𝑁 ⊂ 𝑁 を満たす（つまり終域の制限により左𝐺-加群となる）とき，部分左𝐺-加群((left 𝐺-)submodule)と呼ぶ．

商加群

𝐺を集合，𝑀を左𝐺-加群，𝑁をその部分加群とする．商群𝑀∕𝑁は， ∀ ⁡ 𝑔 ∈ 𝐺 , ∀ ⁡ 𝑥 ∈ 𝑀 , 𝑔 ⁢ ( 𝑥 + 𝑁 ) = 𝑔 ⁢ 𝑥 + 𝑁 をスカラー乗法とする左𝐺-加群をなす．これを商左𝐺-加群と呼ぶ．

商加群の標準射影は準同型

𝐺を集合とする．商左𝐺-加群の標準射影は左𝐺-加群準同型．

同型定理

群の同型定理を引き継いで，加群の準同型定理が成り立つ．

準同型定理

𝐺を集合，𝑀,𝑁を左𝐺-加群， 𝑓 : 𝑀 → 𝑁 を準同型とする． 𝐿 ⊂ ker ⁡ 𝑓 を満たす𝑀の左𝐺-部分加群𝐿と商加群への標準射影 𝑝 : 𝑀 → 𝑀 ∕ 𝐿 に対し，左𝐺-加群準同型 ℎ : 𝑀 ∕ 𝐿 → 𝑁 が存在し， 𝑓 = ℎ ∘ 𝑝 を満たす．

群準同型定理を満たすℎに対して， ℎ ⁡ ( 𝐺 ⁢ 𝑥 ) = 𝐺 ⁢ ℎ ⁡ ( 𝑥 ) を満たすので，示された．

マグマ上の加群

作用と同じく応用上大切なのは，𝐺が何らかの代数的構造を持っている場合．

マグマ上の左加群

𝐺をマグマとする．マグマ上の左𝐺-加群とは，以下の2つのデータからなる．

  • 可換群𝑀
  • スカラー乗法と呼ばれる演算 𝐺 × 𝑀 → 𝑀

これらは以下を満たす．

  • ∀ ⁡ 𝑔 ∈ 𝐺 , ∀ ⁡ 𝑥 , 𝑦 ∈ 𝑀 , 𝑔 ⁢ ( 𝑥 + 𝑦 ) = 𝑔 ⁢ 𝑥 + 𝑔 ⁢ 𝑦
  • ∀ ⁡ 𝑔 , ℎ ∈ 𝐺 , ∀ ⁡ 𝑥 ∈ 𝑀 , 𝑔 ⁢ ( ℎ ⁢ 𝑥 ) = ( 𝑔 ⁢ ℎ ) ⁢ 𝑥

つまり，集合上の𝐺-加群かつマグマ上の𝐺-作用である．

このように，作用同様𝐺の代数的構造をスカラー乗法に写すことが可能．ただし，それだけでなく𝑀の代数的構造に写すことも可能になる．

可換マグマ上の左加群

𝐺を可換マグマとする．可換マグマ上の左𝐺-加群とは，以下の2つのデータからなる．

  • 可換群𝑀
  • スカラー乗法と呼ばれる演算 𝐺 × 𝑀 → 𝑀

これらは以下を満たす．

．
  • ∀ ⁡ 𝑔 ∈ 𝐺 , ∀ ⁡ 𝑥 , 𝑦 ∈ 𝑀 , 𝑔 ⁢ ( 𝑥 + 𝑦 ) = 𝑔 ⁢ 𝑥 + 𝑔 ⁢ 𝑦
  • ∀ ⁡ 𝑔 , ℎ ∈ 𝐺 , ∀ ⁡ 𝑥 ∈ 𝑀 , ( 𝑔 + ℎ ) ⁢ 𝑥 = 𝑔 ⁢ 𝑥 + ℎ ⁢ 𝑥

可換マグマの積 + 𝐺 が可換群の積 + 𝑀 に写される．

性質

特に𝐺が可換群になる場合の性質を見ていこう．

𝐺を可換マグマ，𝑀を左𝐺-加群とする． 𝐺が単位元 0 𝐺 をもつとき，𝑀の単位元 0 𝑀 に対し， 0 𝐺 ⁢ 𝑀 = 0 𝑀

証明
