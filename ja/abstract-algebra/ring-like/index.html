環に似た代数系

環に似た代数系(ring-like structure)とは，以下の3つのデータからなる．

  • 集合𝑅
  • 和と呼ばれる𝑅上の2項演算 + : 𝑅 × 𝑅 → 𝑅
  • 積と呼ばれる𝑅上の2項演算 ⋅ : 𝑅 × 𝑅 → 𝑅

これらは以下の条件を満たす．

  • 𝑅は+を積とするマグマ．
  • 𝑅は⋅を積とするマグマ．
  • ∀ ⁡ 𝑎 , 𝑏 , 𝑐 ∈ 𝑅 , { 𝑎 ⋅ ( 𝑏 + 𝑐 ) = 𝑎 ⋅ 𝑏 + 𝑎 ⋅ 𝑐 ( 𝑎 + 𝑏 ) ⋅ 𝑐 = 𝑎 ⋅ 𝑐 + 𝑏 ⋅ 𝑐

実際によく使うのは，和も積も追加の公理を満たすもの．このとき，和の方がより多くの公理を満たすようにすることが普通．
