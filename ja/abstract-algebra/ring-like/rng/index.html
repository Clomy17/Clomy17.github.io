定義

擬環

擬環(rng, pseudo-ring)とは，以下の3つのデータからなる．

  • 集合𝑅
  • 和と呼ばれる𝑅上の2項演算 + : 𝑅 × 𝑅 → 𝑅
  • 積と呼ばれる𝑅上の2項演算 ⋅ : 𝑅 × 𝑅 → 𝑅

これらは以下の条件を満たす．

  • 𝑅は+を積とする可換群．
  • 𝑅は⋅を積とする半群．
  • ∀ ⁡ 𝑎 , 𝑏 , 𝑐 ∈ 𝑅 , { 𝑎 ⋅ ( 𝑏 + 𝑐 ) = 𝑎 ⋅ 𝑏 + 𝑎 ⋅ 𝑐 ( 𝑎 + 𝑏 ) ⋅ 𝑐 = 𝑎 ⋅ 𝑐 + 𝑏 ⋅ 𝑐

準同型

擬環 𝑅 , 𝑅 ′ を与える．

写像 𝑓 : 𝑅 → 𝑅 ′ が以下を満たすとき，（擬環）準同型((rng) homomorphism)と呼ぶ．

  • ∀ ⁡ 𝑎 , 𝑏 ∈ 𝑅 , 𝑓 ⁡ ( 𝑎 + 𝑏 ) = 𝑓 ⁡ ( 𝑎 ) + 𝑓 ⁡ ( 𝑏 )
  • ∀ ⁡ 𝑎 , 𝑏 ∈ 𝑅 , 𝑓 ⁡ ( 𝑎 ⋅ 𝑏 ) = 𝑓 ⁡ ( 𝑎 ) ⋅ 𝑓 ⁡ ( 𝑏 )

同型

全単射な擬環準同型を同型と呼ぶ．

性質

部分擬環

擬環𝑅を与える．

部分集合𝑆⊂𝑅が𝑅と同じ積で擬環となる，つまり以下を満たすとき，部分擬環(subring)と呼ぶ．

  • 和+に関して𝑆は𝑅の部分（可換）群．つまり以下を満たす．
    • 𝑆 + 𝑆 ⊂ 𝑆
    • 0 𝑅 ∈ 𝑆
    • − 𝑆 = 𝑆
  • 積⋅に関して𝑆は𝑅の部分半群．つまり以下を満たす．
    • 𝑆 ⋅ 𝑆 ⊂ 𝑆
