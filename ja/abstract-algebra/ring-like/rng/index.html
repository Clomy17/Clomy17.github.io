定義

擬環

擬環(rng, pseudo-ring)とは，以下の3つのデータからなる．

  • 集合𝑅
  • 和と呼ばれる𝑅上の2項演算 + : 𝑅 × 𝑅 → 𝑅
  • 積と呼ばれる𝑅上の2項演算 ⋅ : 𝑅 × 𝑅 → 𝑅

これらは以下の条件を満たす．

  • 𝑅は+を積とする可換群．
  • 𝑅は⋅を積とする半群．
  • ∀ ⁡ 𝑎 , 𝑏 , 𝑐 ∈ 𝑅 , { 𝑎 ⋅ ( 𝑏 + 𝑐 ) = 𝑎 ⋅ 𝑏 + 𝑎 ⋅ 𝑐 ( 𝑎 + 𝑏 ) ⋅ 𝑐 = 𝑎 ⋅ 𝑐 + 𝑏 ⋅ 𝑐

擬環が暗に示す公理

擬環𝑅を与えると，以下が成り立つ： 0 ⁢ 𝑅 = { 0 }

証明

∀ ⁡ 𝑥 ∈ 𝑅 , 0 = 0 ⁢ 𝑥 − 0 ⁢ 𝑥 = ( 0 + 0 ) ⁢ 𝑥 − 0 ⁢ 𝑥 = 0 ⁢ 𝑥

準同型

擬環 𝑅 , 𝑅 ′ を与える．

写像 𝑓 : 𝑅 → 𝑅 ′ が以下を満たすとき，（擬環）準同型((rng) homomorphism)と呼ぶ．

  • 和に関して可換群準同型： ∀ ⁡ 𝑎 , 𝑏 ∈ 𝑅 , 𝑓 ⁡ ( 𝑎 + 𝑏 ) = 𝑓 ⁡ ( 𝑎 ) + 𝑓 ⁡ ( 𝑏 )
  • 積に関して半群準同型： ∀ ⁡ 𝑎 , 𝑏 ∈ 𝑅 , 𝑓 ⁡ ( 𝑎 ⋅ 𝑏 ) = 𝑓 ⁡ ( 𝑎 ) ⋅ 𝑓 ⁡ ( 𝑏 )

準同型が暗に示す公理より直ちに以下が成り立つ：

準同型が暗に示す公理

  • 擬環𝑅,𝑅′
  • 準同型𝑓:𝑅→𝑅′

を与えると，以下が成り立つ：

  • 和の単位元を保つ： 𝑓 ⁡ ( 0 𝑅 ) = 0 𝑅 ′
  • 和の逆元を保つ： ∀ ⁡ 𝑥 ∈ 𝑅 , 𝑓 ⁡ ( − 𝑥 ) = − 𝑓 ⁡ ( 𝑥 )

同型

全単射な擬環準同型を（擬環）同型((rng) isomorphism)と呼ぶ．

擬環の圏

  • 対象を全ての擬環
  • 射を全ての擬環準同型

とする圏を擬環の圏(category of rngs)と呼ぶ

同型定理

擬環の圏の部分対象，商対象を確認し，同型定理を示す．

部分擬環

擬環𝑅を与える．

部分集合𝑆⊂𝑅が𝑅と同じ和と積に関して擬環となる，つまり以下を満たすとき，部分擬環(trivial subrng)と呼ぶ．

  • 和+に関して𝑆は𝑅の部分可換群．つまり以下を満たす．
    • 𝑆 + 𝑆 ⊂ 𝑆
    • 0 𝑅 ∈ 𝑆
    • − 𝑆 = 𝑆
  • 積⋅に関して𝑆は𝑅の部分半群．つまり以下を満たす．
    • 𝑆 ⋅ 𝑆 ⊂ 𝑆

自明な部分擬環

擬環𝑅を与える．

  • 𝑅自身は𝑅の部分擬環である．これを𝑅の自明な部分擬環(trivial subrng)と呼ぶ．
  • 𝑅以外の部分擬環を非自明(nontrivial)や真(proper)と呼ぶ．
