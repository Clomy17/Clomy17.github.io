左準群

左準群(left quasigroup)とは，以下の3つのデータからなる．

  • 集合𝐺
  • 積と呼ばれる𝐺の2項演算 ⋅ : 𝐺 × 𝐺 → 𝐺
  • 左除法と呼ばれる𝐺上の2項演算 ∕ : 𝐺 × 𝐺 → 𝐺

これらは以下の条件を満たす．

  • ∀ ⁡ 𝑎 , 𝑏 ∈ 𝐺 , ( 𝑏 ⁢ 𝑎 ) ∕ 𝑎 = ( 𝑏 ∕ 𝑎 ) ⁢ 𝑎 = 𝑏

右準群

右準群(right quasi group)とは，以下の3つのデータからなる．

  • 集合𝐺
  • 積と呼ばれる𝐺の2項演算 ⋅ : 𝐺 × 𝐺 → 𝐺
  • 右除法と呼ばれる𝐺上の2項演算 ∖ : 𝐺 × 𝐺 → 𝐺

これらは以下の条件を満たす．

  • ∀ ⁡ 𝑎 , 𝑏 ∈ 𝐺 , 𝑎 ∖ ( 𝑎 ⁢ 𝑏 ) = 𝑎 ⁢ ( 𝑎 ∖ 𝑏 ) = 𝑏

準群(quasi group)とは，以下の4つのデータからなる．

  • 集合𝐺
  • 積と呼ばれる𝐺の2項演算 ⋅ : 𝐺 × 𝐺 → 𝐺
  • 左除法と呼ばれる𝐺上の2項演算 ∕ : 𝐺 × 𝐺 → 𝐺
  • 右除法と呼ばれる𝐺上の2項演算 ∖ : 𝐺 × 𝐺 → 𝐺

これらは以下の条件を満たす．

  • ∀ ⁡ 𝑎 , 𝑏 ∈ 𝐺 , ( 𝑏 ⁢ 𝑎 ) ∕ 𝑎 = ( 𝑏 ∕ 𝑎 ) ⁢ 𝑎 = 𝑏
  • ∀ ⁡ 𝑎 , 𝑏 ∈ 𝐺 , 𝑎 ∖ ( 𝑎 ⁢ 𝑏 ) = 𝑎 ⁢ ( 𝑎 ∖ 𝑏 ) = 𝑏

Wikipediaと左右が逆だが，単位元𝑒を持ったときに，𝑒∕𝑎が𝑎の左逆元を与えて欲しかったためにこうした．

準同型

𝐺 , 𝐺 ′ を準群とする．写像 𝑓 : 𝐺 → 𝐺 ′ が以下を満たすとき，（準群）準同型と呼ぶ．

∀ ⁡ 𝑔 , ℎ ∈ 𝐺 , 𝑓 ⁡ ( 𝑔 ⁢ ℎ ) = 𝑓 ⁡ ( 𝑔 ) ⁢ 𝑓 ⁡ ( ℎ )

    • 左準群𝐺,𝐻
    • 準同型 𝑓 : 𝐺 → 𝐻

    • 準同型 𝑓 : 𝐺 → 𝐻
    を与えると，以下を満たす： ∀ ⁡ 𝑎 , 𝑏 ∈ 𝐺 , 𝑓 ⁡ ( 𝑎 ∖ 𝑏 ) = 𝑓 ⁡ ( 𝑎 ) ∖ 𝑓 ⁡ ( 𝑏 )
