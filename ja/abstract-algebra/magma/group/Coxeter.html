Coxeter群

Coxeter群(Coxeter group)とは，以下の2つのデータからなる．

  • 群𝑊
  • 𝑊の生成系𝑆⊂𝑊

これらは，適当な 𝑋 ⊂ 𝑆 × 𝑆 ∖ { ( 𝑎 , 𝑎 ) | 𝑎 ∈ 𝑆 } と 𝑚 𝑠 , 𝑡 : 𝑋 → ℤ ≥ 2 を用いて定まる関係𝑅により， 𝑅 ≔ { 𝑠 2 ∈ 𝑊 | 𝑠 ∈ 𝑆 } ∪ { ( 𝑠 ⁢ 𝑡 ) 𝑚 𝑠 , 𝑡 ∈ 𝑊 | ( 𝑠 , 𝑡 ) ∈ 𝑋 } を用いて ⟨ 𝑊 | 𝑅 ⟩ と表示される．

𝑊のこともCoxeter群と呼ぶ．

𝑊をCoxeter群，𝑆をその生成系とする．

∀ ⁡ 𝑠 , 𝑡 ∈ 𝑆 に対し，𝑠⁢𝑡の位数を返す二項演算 𝑚 𝑠 , 𝑡 : 𝑆 × 𝑆 → ℕ ∪ { ∞ } をCoxeter行列と呼ぶ．ただし，位数の場合は0の代わりに∞とする．
