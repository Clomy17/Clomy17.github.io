イデアル・フィルター

下方集合

前順序集合 (𝑃,≤) を与える．

  • 下方集合(lower set, down-set)とは，𝑃 の部分集合 𝐷⊂𝑃 であって，以下を満たすもののこと： ∀ ⁡ 𝑥∈𝐷 , ∀ ⁡ 𝑦∈𝑃 , 𝑦≤𝑥 ⇒ 𝑦∈𝐷
  • 上方集合(upper set, up-set)とは，𝑃 の部分集合 𝑈⊂𝑃 であって，以下を満たすもののこと： ∀ ⁡ 𝑥∈𝑈 , ∀ ⁡ 𝑦∈𝑃 , 𝑦≥𝑥 ⇒ 𝑦∈𝑈

有限有向集合

  • 半順序集合 (𝑃,≤) が，有限（上方）有向集合(finitely (upward-)directed set)であるとは，以下の同値な条件のいずれか（ゆえに全て）を満たすことを言う：
    • ∀ ⁡ 𝑎,𝑏 ∈ 𝑃 に対し， { 𝑎,𝑏 } ⊂ 𝑃 が上界を持つ．
    • 任意の 𝑃 の有限部分集合が上界を持つ．
  • 半順序集合 (𝑃,≤) が，有限下方有向集合(finitely downwarddirected set)であるとは，以下の同値な条件のいずれか（ゆえに全て）を満たすことを言う：
    • ∀ ⁡ 𝑎,𝑏 ∈ 𝑃 に対し， { 𝑎,𝑏 } ⊂ 𝑃 が下界を持つことを言う．
    • ∀ ⁡ 𝑎,𝑏 ∈ 𝑃 に対し， { 𝑎,𝑏 } ⊂ 𝑃 が下界を持つ．
    • 任意の 𝑃 の有限部分集合が上界を持つ．

イデアル・フィルター

前順序集合 (𝑃,≤) を与える．

  • 𝑃 のイデアル(ideal)とは，非空な 𝑃 の部分集合であって，下方集合かつ有限有向集合となるものを言う．
  • 𝑃 のフィルター(filter)とは，非空な 𝑃 の部分集合であって，上方集合かつ有限下方有向集合となるものを言う．

下方閉包・上方閉包

  • 半順序集合 (𝑃,≤)
  • 𝑃 の部分集合 𝑆⊂𝑃

を与える．

  • ↓⁡𝑆⊂𝑃 が 𝑆 の下方閉包(down closure)であるとは，以下の同値な条件のいずれか（ゆえに全て）を満たすことを言う：
    • 𝑆 ⊂ ↓⁡𝑆 なる下方集合のうち，最小のもの．
    • ↓⁡𝑆 = { 𝑥∈𝑃 | ∃ ⁡ 𝑠∈𝑆 , 𝑥≤𝑠 }
  • ↑⁡𝑆⊂𝑃 が 𝑆 の上方閉包(up closure)であるとは，以下の同値な条件のいずれか（ゆえに全て）を満たすことを言う：
    • 𝑆 ⊂ ↑⁡𝑆 なる上方集合のうち，最小のもの．
    • ↑⁡𝑆 = { 𝑥∈𝑃 | ∃ ⁡ 𝑠∈𝑆 , 𝑠≤𝑥 }

単項イデアル・単項フィルター

  • 半順序集合 (𝑃,≤)
  • 𝑎∈𝑃

を与える．

  • ↓⁡𝑎⊂𝑃 が 𝑎 で生成される単項イデアル(principal ideal generated by 𝑎)であるとは，以下の同値な条件のいずれか（ゆえに全て）を満たすことを言う：
    • {𝑎} ⊂ ↓⁡𝑎 なるイデアルのうち，最小のもの．
    • {𝑎} の下方閉包： ↓⁡𝑎 = ↓ ⁡ {𝑎} = { 𝑥∈𝑃 | 𝑥≤𝑎 }
  • ↑⁡𝑎⊂𝑃 が 𝑎 で生成される単項フィルター(principal filter generated by 𝑎)であるとは，以下の同値な条件のいずれか（ゆえに全て）を満たすことを言う：
    • {𝑎} ⊂ ↑⁡𝑎 なるフィルターのうち，最小のもの．
    • {𝑎} の上方閉包： ↑⁡𝑎 = ↑ ⁡ {𝑎} = { 𝑥∈𝑃 | 𝑎≤𝑥 }
半順序の包含による表現定理
  • 半順序集合 (𝑃,≤)
  • 𝑃 の冪集合を包含関係により半順序集合見做したもの (2𝑃,≤)

を与えると，以下が成り立つ：

  • 単項イデアルを取る操作 ↓ : 𝑃 → 2𝑃 は関係埋め込み．特に， ↓ : 𝑃 → im ⁡ (↓) は関係同型．
  • 単項フィルターを取る操作 ↑ : 𝑃 → 2𝑃 は関係逆埋め込み．特に， ↑ : 𝑃 → im ⁡ (↑) は関係逆同型．

証明

  • ∀ ⁡ 𝑎,𝑏,𝑥 ∈ 𝑃 に対し，≤ の推移律より 𝑎≤𝑏 ⇒ ( 𝑥≤𝑎 ⇒ 𝑥≤𝑏 ) ⇒ ↓⁡𝑎 ⊆ ↓⁡𝑏
  • ∀ ⁡ 𝑎,𝑏,𝑥 ∈ 𝑃 に対し，≤ の反射律より， 𝑎 ∈ ↓⁡𝑎 なので， ↓⁡𝑎 ⊆ ↓⁡𝑏 ⇒ 𝑎 ∈ ↓⁡𝑏 ⇒ 𝑎≤𝑏

以上より，↓は関係埋め込み．

を与える．

分配束()とは，以下のデータのこと：

昇鎖条件・降鎖条件

昇鎖・降鎖

  • 半順序集合の昇鎖(ascending chain) とは，最小元を持つ鎖のこと．
  • 半順序集合の降鎖(descending chain) とは，最大元を持つ鎖のこと．

昇鎖条件・降鎖条件

  • 半順序集合が昇鎖条件(ACC (ascending chain condition)) を満たすとは，任意の昇鎖が有限長となることを言う．
  • 半順序集合が降鎖条件(ACC (descending chain condition)) を満たすとは，任意の降鎖が有限長となることを言う．

昇鎖条件と整礎関係は同値

半順序集合 ( 𝑃,≤ ) を与えると，（従属選択公理において）以下を満たす：

  • ( 𝑃,≤ ) が昇鎖条件を満たすことと，余整礎集合であることは同値．
  • ( 𝑃,≤ ) が降鎖条件を満たすことと，整礎集合であることは同値．

昇鎖条件と有限性の関係

  • 無限長の鎖を持つ半順序集合は無限長の昇鎖または降鎖を持つ．
  • 無限半順序集合は以下のいずれかを持つ：
    • 無限幅の反鎖
    • 無限長の昇鎖
    • 無限長の降鎖
  • 半順序集合に対し，以下は同値：
    • 有限
    • 無限長の鎖を持たない
    • 無限幅の反鎖を持たない

束の昇鎖条件

  • 降鎖条件を満たす束 ( 𝑃,≤,∧,∨ ) に対し，任意の空でない部分集合 𝑆⊆𝑃 に対し，ある有限部分集合 𝑆0 ⊆ 𝑆 が存在し，𝑆 の下限は 𝑆0 の下限に等しい： ⋀⁡𝑆 = ⋀ ⁡ 𝑆0
  • 最大元を持つ，降鎖条件を満たす束は，完備束．
  • 昇鎖条件を満たす束 ( 𝑃,≤,∧,∨ ) に対し，任意の空でない部分集合 𝑆⊆𝑃 に対し，ある有限部分集合 𝑆0 ⊆ 𝑆 が存在し，𝑆 の上限は 𝑆0 の上限に等しい： ⋁⁡𝑆 = ⋁ ⁡ 𝑆0
  • 最小元を持つ，昇鎖条件を満たす束は，完備束．

証明

𝑆 の全ての有限部分集合の下限の集合は，𝑃 の空でない部分集合なので，極小元を持ち，それは有限部分集合 𝑆0 ⊂ 𝑆 を用いて， ⋀ ⁡ 𝑆0 と表せる． ∀ ⁡ 𝑎∈𝑆 に対し， ⋀ ⁡ ( 𝑆0 ∪ {𝑎} ) = ( ⋀ ⁡ 𝑆0 ) ∧ 𝑎 ≤ ⋀ ⁡ 𝑆0 を満たすが， ⋀ ⁡ 𝑆0 の極小性より， ⋀ ⁡ ( 𝑆0 ∪ {𝑎} ) = ⋀ ⁡ 𝑆0 となる．よって， ⋀ ⁡ 𝑆0 ≤ 𝑎 なので， ⋀ ⁡ 𝑆0 は 𝑆 の下界．これは自明に下限となる．

  • 下半束 ( 𝑃,≤,∧ ) を与える．
    • 𝑆⊂𝑃 が下限素または交叉素(meet-prime)であるとは，以下を満たすことを言う： ∀ ⁡ 𝑎,𝑏 ∈ 𝑆 , 𝑎∧𝑏 ∈ 𝑆 ⇒ 𝑎∈𝑆 ∪ 𝑏∈𝑆
    • 𝑥∈𝑃 が下限素または交叉素(meet-prime)であるとは，以下の同値な条件のいずれか（ゆえに全て）を言う：
      • ∀ ⁡ 𝑎,𝑏 ∈ 𝑆 , 𝑎∧𝑏 ≤ 𝑥 ⇒ 𝑎≤𝑥 ∪ 𝑏≤𝑥
